This Is Simple Pagination Example
Code: Using data-pages attribute 
<div id="pagination" data-pages='{"page":1,"pages":10,"linkscount":3}'></div>
<button onclick='$("#pagination").paginate();'>show Pagination</button>
here goes pagination



Code: Using parameters passed(linkscount not available) 
<div id="pagination_1" ></div>
<button onclick='$("#pagination").paginate(1, 10);'>show Pagination</button>
here goes pagination



Code: Using Object parameter (page and pages properties are required) 
<div id="pagination_2" ></div>
<button onclick='$("#pagination_2").paginate({page:1, pages:10, linkscount: 3});'>show Pagination</button>
here goes pagination
